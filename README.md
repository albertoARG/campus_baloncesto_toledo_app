# Campus Baloncesto App 🏀

Aplicación multiplataforma (Web y Móvil) desarrollada con **Flutter** para la gestión integral de campus de baloncesto. Este proyecto ha sido diseñado como parte de un Trabajo de Fin de Grado (TFG) para automatizar y optimizar las tareas de entrenadores, coordinadores y jugadores durante el transcurso de un campus deportivo.

## 🚀 Características Principales

*   **Gestión de Grupos e Integrantes:** Organización de entrenadores y jugadores en grupos de entrenamiento.
*   **Generador Automático de Equipos:** Algoritmo táctico inteligente que genera equipos equilibrados en función de la edad, posición de juego y nivel técnico de los jugadores.
*   **Seguimiento Estadístico:** Registro de puntuaciones de tiro libre, competiciones y evolución diaria de cada jugador.
*   **Notificaciones Push en Tiempo Real:** Integración con Firebase Cloud Messaging para alertas instantáneas de horarios, veladas y novedades del campus.
*   **Panel de Administración:** Control total sobre el registro de usuarios, roles (administrador, entrenador, jugador) y parametrización de actividades.

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** Flutter & Dart
*   **Backend & Base de Datos:** Supabase (PostgreSQL)
*   **Notificaciones:** Firebase Cloud Messaging (FCM)
*   **Almacenamiento de Multimedia:** Cloudinary (para imágenes de perfil y galería)
*   **Despliegue Web:** Firebase Hosting
